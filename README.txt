###Installing required packages
Required windows installation are included in the pwizLibraries-and-Installation folder. If you have 32-bit Windows OS click 32-Bit_Double-Click_To_Install.bat, if 64-bit click 64-Bit_Double-Click_To_Install.bat.

###Converting files
To convert all files, put raw files without folder structure inside Input_Folder.

Double click on Double-Click_To-Convert.bat. Converted files will be placed in Output_Folder. The log output will be placed in log.txt.

The Input_Folder as part of the download contains example files from a variety of vendors.
